The program can be executed using command 'python3 filename.py <argv 1>' in the terminal

Currently at default, the program will read 100 images from each classes, but it might be too long for testing purpose. Thus user can type 
the argument in <argv1>, the number inside will signifies the number of images user wants to read from each class.
The input at <argv1> must stricly be an integers, or the program will crash

After the program is run, the result will be printed at the terminal, with labels showing which result it is.






Users need to have SciKit Learn, Numpy and openCV installed on their laptop to be able to run the code.
